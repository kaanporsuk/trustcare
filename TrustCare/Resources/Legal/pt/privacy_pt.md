# Politica de Privacidade

## Dados que recolhemos
- Recolhemos os dados da conta fornecidos durante o registo.
- Armazenamos dados de perfil como nome, idioma e preferencias.
- Podemos processar interacoes com prestadores e itens guardados.
- Registos tecnicos podem ser usados para melhorar fiabilidade e seguranca.

## Como usamos os dados
- Os dados sao usados para disponibilizar funcionalidades principais do TrustCare.
- Usamos informacao para personalizar idioma e conteudo.
- Analises de utilizacao ajudam a melhorar a qualidade do produto.

## Partilha de dados
- Nao vendemos os seus dados pessoais.
- Os dados podem ser partilhados com prestadores de servico necessarios para operar a aplicacao.
- Podemos divulgar dados quando exigido por lei.

## Os seus direitos
- Pode solicitar acesso aos seus dados pessoais.
- Pode solicitar correcao de informacoes de perfil incorretas.
- Pode solicitar eliminacao da conta e dos dados relacionados.
- Pode contactar o suporte para pedidos relacionados com privacidade.

## Cookies e tecnologias semelhantes
- Podemos usar armazenamento local para definicoes e continuidade de sessao.
- Estas tecnologias ajudam a manter autenticacao segura.
- Pode limpar os dados da aplicacao ao terminar sessao e remover dados locais.

## Seguranca
- Aplicamos medidas para proteger dados pessoais.
- Nenhum sistema e perfeito, mas melhoramos continuamente as praticas de seguranca.
- Reporte imediatamente ao suporte qualquer problema de seguranca suspeito.

## Contacto
- Contacte o suporte para qualquer questao sobre privacidade.
- As atualizacoes de politica podem aparecer nesta secao.
