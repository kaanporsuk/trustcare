# Integritetspolicy

## Data vi samlar in
- Vi samlar in kontouppgifter som du anger vid registrering.
- Vi lagrar profildata som namn, sprak och preferenser.
- Vi kan behandla interaktioner med vardgivare och sparade objekt.
- Tekniska loggar kan anvandas for att forbattra tillforlitlighet och sakerhet.

## Hur vi anvander data
- Data anvands for att leverera TrustCares huvudfunktioner.
- Vi anvander information for att anpassa sprak och innehall.
- Anvandarinsikter hjalper oss att forbattra produktkvaliteten.

## Datadelning
- Vi saljer inte dina personuppgifter.
- Data kan delas med tjansteleverantorer som behovs for att driva appen.
- Vi kan lamna ut data nar lag krav staller det.

## Dina rattigheter
- Du kan begara tillgang till dina personuppgifter.
- Du kan begara korrigering av felaktig profilinformation.
- Du kan begara radering av ditt konto och relaterade data.
- Du kan kontakta support for integritetsrelaterade fragor.

## Cookies och liknande tekniker
- Vi kan anvanda lokal lagring for installningar och sessionskontinuitet.
- Dessa tekniker hjalper till att uppratthalla saker autentisering.
- Du kan rensa appdata genom att logga ut och ta bort lokal data.

## Sakerhet
- Vi anvander skyddsatgarder for att skydda personuppgifter.
- Inget system ar perfekt, men vi forbattrar kontinuerligt vara sakerhetsrutiner.
- Rapportera misstankta sakerhetsproblem till support omgaende.

## Kontakt
- Kontakta support vid fragor om integritet.
- Policyuppdateringar kan visas i denna sektion.
