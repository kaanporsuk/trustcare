# Politique de confidentialité

## Données que nous collectons
- Nous collectons les informations de compte fournies lors de l'inscription.
- Nous stockons des données de profil telles que le nom, la langue et les préférences.
- Nous pouvons traiter les interactions avec les prestataires et les éléments enregistrés.
- Des journaux techniques peuvent être utilisés pour améliorer la fiabilité et la sécurité.

## Comment nous utilisons les données
- Les données sont utilisées pour fournir les fonctionnalités principales de TrustCare.
- Nous utilisons les informations pour personnaliser la langue et le contenu.
- Les analyses d'usage nous aident à améliorer la qualité du produit.

## Partage des données
- Nous ne vendons pas vos données personnelles.
- Les données peuvent être partagées avec les prestataires nécessaires au fonctionnement de l'application.
- Nous pouvons divulguer des données lorsque la loi l'exige.

## Vos droits
- Vous pouvez demander l'accès à vos données personnelles.
- Vous pouvez demander la correction d'informations de profil inexactes.
- Vous pouvez demander la suppression de votre compte et des données associées.
- Vous pouvez contacter le support pour les demandes liées à la confidentialité.

## Cookies et technologies similaires
- Nous pouvons utiliser le stockage local pour les réglages et la continuité de session.
- Ces technologies aident à maintenir une authentification sécurisée.
- Vous pouvez effacer les données de l'application en vous déconnectant et en supprimant les données locales.

## Sécurité
- Nous appliquons des mesures pour protéger les données personnelles.
- Aucun système n'est parfait, mais nous améliorons continuellement nos pratiques de sécurité.
- Signalez rapidement tout problème de sécurité suspect au support.

## Contact
- Contactez le support pour toute question liée à la confidentialité.
- Les mises à jour de la politique peuvent apparaître dans cette section.
